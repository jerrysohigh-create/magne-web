Security
MAGNE.AI features a dedicated secure element that isolates your private keys from the main operating system. This hardware-level isolation is designed to help protect sensitive keys and signing flows, even under elevated device-risk scenarios. The secure element handles all signing operations without exposing keys to the main processor.
Security is our top priority. MAGNE.AI implements multiple layers of protection: hardware-level encryption via secure element, biometric authentication (fingerprint + face), encrypted storage, and secure boot chain. Your private keys never leave the secure hardware, providing hardware wallet-grade security architecture for your digital assets.
Your assets can be recovered using your backup phrase (seed phrase) which you should securely store when setting up your wallet. We recommend storing your backup phrase offline in a safe location. With your backup phrase, you can restore access to your assets on a new device.
Web3 & Blockchain
MAGNE.AI supports native dual-chain architecture (L1 + L2), enabling interaction with multiple blockchain networks. The device is optimized for major ecosystems including Ethereum, BNB Chain, Polygon, and various Layer 2 solutions. Check our Web3 Network section for the complete list of supported chains.
