AI Capabilities

On-Device AI Translation + Gemini Cloud Intelligence

MAGNE.AI Phone Gen1 combines local AI translation capabilities with Android-native Gemini cloud intelligence. The device-side AI stack is designed to support Chinese-English call translation, while Gemini provides broader cloud-assisted AI features through the Android ecosystem.

On-Device Call Translation

The primary on-device AI capability of MAGNE.AI Phone Gen1 is Chinese-English call translation. During supported call scenarios, speech is converted into text through an ASR model, processed by a local language model, and presented through the call translation application.

Local Model Stack

MAGNE.AI Phone Gen1 uses a device-side Qwen 1.5B-class language model for selected translation workflows. Running the translation model on the terminal helps reduce round-trip latency and supports use cases where responsiveness is important.

Real-time Translation

Selected AI workflows can run on the device, helping improve responsiveness and reduce reliance on cloud processing in privacy-sensitive scenarios.

Low Latency

Local model inference is designed to support real-time translation scenarios where network round-trip delays may impact user experience.

Privacy-Sensitive

Translation processing on-device means sensitive conversation content may remain on the device, reducing exposure to cloud-based processing.

Selected Workflows

On-device AI is designed for specific translation scenarios. Broader AI capabilities are supported through cloud-based services where available.

Speech-to-Text Pipeline

The call translation workflow includes an open-source ASR model used to convert speech into text. The translated result is then displayed in the call translation application, creating a practical communication workflow for bilingual scenarios.

Open-Source ASR

The speech-to-text pipeline uses an open-source ASR model, converting spoken language into text for further processing by the local translation model.

Bilingual Support

Designed to support Chinese-English translation scenarios, helping enable communication across language barriers in call environments.

Application Integration

Translation results are presented through a dedicated call translation application, providing a practical user interface for real-world communication needs.
