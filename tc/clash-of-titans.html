1. 世界觀與願景

在時間的盡頭，舊神的秩序崩塌。光明與黑暗在混沌中重生，諸神遺物散落於世界各地。獲得遺物庇護者，將有機會超越命運，挑戰諸神。

Clash of the Titans 講述人類戰士重奪神力的史詩故事。玩家將成為「被選中的英雄」，在神域廢墟中重建秩序，收集遺物、鍛造裝備、銘刻符文，征服迷宮與競技場，並逐步喚醒自身神力。

🎯 核心理念

玩法機制與鏈上互動被設計為支援生態內的沉浸式策略參與。Clash of the Titans 不僅是一款策略戰鬥遊戲，也是一個動態平衡的成長型生態系統。每一次戰鬥、精煉與決策，都會影響遊戲內資源流動與經濟平衡。

核心原則

  • 所有權：玩家擁有其資產與成長成果
  • 參與度：戰鬥與成長形成正向循環
  • 平衡性：消耗與回收維持資源健康

💎 遺物系統

遺物被設計為可與遊戲內經濟系統互動，包括可能基於 MHA 的購買或接入機制；具體形式須以最終遊戲設計及官方發布條款為準。遺物決定角色的核心能力，並由神力值與品質係數共同決定：

屬性 = 基礎屬性 ×（神力值 ÷ 100）× 品質係數

品質神力係數名稱

🔬 精煉系統

當神力值 ≤ 160 時，玩家可使用 PRIMORDIUM（PRI）進行精煉：

等級神力PRI 消耗
基礎125–1403 PRI
中級141–1509 PRI
高級151–16015 PRI

精煉具有固定機率分布——神力值越高，成功機率越低。此機制是 PRI 消耗與遺物價值成長的主要途徑。

⚡ 強化系統

強化使用強化石進行。強化等級越高所需石頭越多，但提供穩定的屬性提升。強化成功率隨強化等級提升而下降。

4. 技術框架

⛓️ 區塊鏈網絡

Clash of the Titans 被設計為橫跨 MAGNE L1 及 MHash L2 的應用展示案例，將遊戲玩法、錢包互動、NFT 類資產與鏈上結算流程對齊 MAGNE Network 架構。

模組技術基礎
網絡架構MAGNE L1 + MHash L2
智能合約標準ERC-721（NFT）、ERC-20（MHA）、ERC-1155（NFT）
數據存儲分散式鏈上索引（IPFS）
錢包支援MetaMask / Rabby / Magne Wallet
跨鏈橋BSC、Polygon 兼容轉帳

Magne 網絡同時作為遊戲執行層與經濟結算層：

  • 遊戲邏輯在中心化伺服器運行
  • 資產交易、獎勵及 NFT 元數據在鏈上
  • 所有玩家 NFT 及代幣可在鏈上驗證與追溯

📜 智能合約結構

模組功能
RelicContract記錄遺物 NFT 屬性及強化狀態
EquipContract裝備 NFT 生成、合成及分解邏輯
InscriptionContract銘刻 NFT 存儲及套裝效果觸發
TreasuryContract管理 MHA 與 PRI 消耗、回收及池化邏輯
RewardContract管理符合條件的競技場與迷宮獎勵邏輯，具體須以最終遊戲規則及官方發布條款為準。

🔒 安全機制

  • 多重簽署治理用於提升合約升級安全性
  • 鏈上核驗有助於防止偽造資產
  • 所有資產交易均可透過區塊鏈瀏覽器核驗
  • 定期審計與緊急回滾策略用於支援公平性

🛡️ 數據與資產安全

  • 玩家資產由個人鏈上錢包持有，不進行中心化鎖定
  • 所有 NFT 屬性及成長記錄可驗證及追溯
  • 所有 PRI 與 MHA 流轉均公開記錄於鏈上
  • 遊戲帳號僅作為互動中介

6. 願景

打造一個由「神話敘事 + 區塊鏈成長」構建的可持續遊戲經濟。

Clash of the Titans 不僅是策略與榮耀的競爭，也是一個持續演進的數字經濟。每一次戰鬥與 NFT 類互動，都被設計為對遊戲內經濟、文化敘事及鏈上活動模型產生貢獻。

在Clash of the Titans 的世界中，諸神之力不再由天界壟斷。每位玩家都是命運的繼承者與力量的重鑄者。他們以遺物為媒介、以戰鬥為證明，在永恆神域中不斷追求更高秩序。
